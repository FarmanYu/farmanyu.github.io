##git-tf使用简单说明
###git tf clone
clone tfs代码到本地
	
	git tf clone [tfs host + tfs product] [tfs path]
	git tf clone http://xx.xx.x.x:8080/tfs/Vacations/ $/Cruise/Release/162_0812/Cruise.Mobile/WebAppSEO5.8/WebApp

###git tf checkin
checkin代码到tfs, 必须先将本地代码提交.

	git tf clone

###git tf pull
更新tfs代码到本地
	
	git tf pull

##多人开发

1.先创建本地dev分支	

	git checkout -b dev

2.将代码提交本地dev分支
	
	git commit -am "commit code"

3.切换到主分支
	
	git checkout master

4.更新代码
	
	git tf pull

5.合并分支代码
	
	git merge dev

6.编译代码
	
	grunt web:cruise

7.提交编译代码
	
	git commit -am "build code"

8.checkin代码
	
	git tf checkin
